# Orszag-Tang

Simulate the Orszag-Tang vortex (2D MHD equations)

Philip Mocz (2025)

Usage:

```console
python orszag_tang.py
```

Takes around 55 seconds ('llf') / 105 seconds ('hlld') to run on my macbook (cpu).
